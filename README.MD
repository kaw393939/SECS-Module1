# Module Objective
You will be able to setup and configure Visual Studio 2019 to write C# programs that use Docker, GIT, and GitHub
# Readings
* None

# Download and Install
* [Download and Install Visual Studio 2019 Community Edition](https://visualstudio.microsoft.com/vs/)
## Prerequisites
* [Docker Desktop Windows](https://www.docker.com/)
* [GIT SCM](https://git-scm.com/download/win)

# Hands On Assignment
Download and install Visual Studio 2019 and create a C# “hello World that runs within a Docker container.  Then push your project to GitHub and submit it on Moodle.

Create a dockerized C# hello world application and push publish it on GitHub.

## Submission Instructions
Please submit to the learning management system(Moodle) a link to the GitHub repository for this assignment.
